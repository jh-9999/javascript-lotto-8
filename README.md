# 로또 발매기 (CLI)

간단한 콘솔 기반 로또 발매기 입니다.
**구매 금액을 입력**하면 금액/1,000 만큼 로또 티켓을 발행하고, 
**당첨 번호/보너스 번호**를 기준으로 등수 통계를 계산하여 **수익률**을 출력합니다.


## 주요 기능 
#### 입력/출력
- 사용자 입력 받기
- 메시지 출력하기

#### 입력값 공통 검증
- 빈값(공백 포함) 여부 확인

#### 구입 금액 처리
- 숫자만 입력 허용
- 1,000원 단위 확인
- 최소 1,000원 이상 확인
- 입력 금액을 구매 수량으로 환산

#### 로또 번호 생성
- 1 ~ 45 범위에서 중복 없이 숫자 6개 랜덤 추출
- 오름차순 정렬 및 출력

#### 당첨번호 입력 처리
- 숫자와 쉼표(,)만 입력 허용
- 콤마로 분리 후 빈 항목 제거
- 문자열을 숫자 배열로 변환
- 숫자 개수 6개 확인
- 각 숫자가 정수 1 ~ 45 범위인지 확인

#### Lotto 클래스
- 입력 숫자 개수(6개) 검증 예외 처리
- 중복 숫자 존재 여부 검증 예외 처리
- 현재 번호 배열 반환
- 당첨 번호와 일치 개수 계산
- 보너스 번호 포함 여부 확인

#### 보너스 번호 입력 처리
- 1개 입력 여부 확인
- 당첨 번호와 중복되지 않는지 확인
- 1 ~ 45 범위 확인

#### 결과 및 수익률
- 등수별 당첨 개수 집계 및 출력
- 총 수익률 계산 및 출력

#### 에러 처리/재입력
- 잘못된 입력 시 에러 메시지 출력 후 재입력 루프
